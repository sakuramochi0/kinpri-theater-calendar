# kinpri-theater-calendar

📆 キンプリの映画の上映スケジュールをiCalとして生成するプログラム。Googleカレンダーなどのカレンダーアプリに登録すると自動更新されて最新のスケジュールが確認できます。

何らかの理由でカレンダーが正しく更新されない場合もあるので、予約前にリンクから映画館のスケジュールページも確認するようにしてください。

## 現在の対応映画館
